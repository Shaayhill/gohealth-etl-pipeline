🏥 GoHealth Data Engineering Project

Hello! I’m Shaayla, and this is a real-world EMR data pipeline I built as part of a technical data engineering assessment. The project uses Python and pandas to process patient, visit, lab result, and ICD datasets from a structured Excel file.

📌 What It Does

  • Ingests data from four Excel tabs
  • Cleans and joins multiple datasets
  • Calculates insights like average visits and top ICD codes
  • Outputs a summarized CSV report
